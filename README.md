# ESP-Watch-OBD2
OBD2 gauge using Lilygo ESP32 watch bluetooth and ELM327 bluetooth OBD2 dongle 

This project uses ELMduino and TTGO library to connect to a bluetooth OBD2 ELM327 dongle "OBDII".
